2. Genera uno con permiso repo
3. Pega el token aquí (se guarda solo en tu dispositivo)

Publicar manualmente

Tu navegador no puede conectar con la API de GitHub. Usa este método alternativo:

Pasos:
1. Descarga el JSON con el botón de arriba
2. Ve a tu repo en GitHub → Add file → Upload files
3. Arrastra el .json descargado y haz commit
4. El modelo GLB ya está en el repo ✅
